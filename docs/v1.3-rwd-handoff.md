# v1.3 第 13.1 節 RWD、Dark Mode 與操作語意交接

日期：2026-07-16
狀態：第 13.1 節工程實作與瀏覽器驗證完成；第 14 節完整業務 manual smoke 尚待使用者執行。

## 本次完成

- `AppLayout` 在小於 `lg` 的尺寸改用 overlay drawer，桌機維持 224px 固定 Sidebar；手機可由 Header 開啟、遮罩或關閉按鈕收合，換頁後自動收合。
- 主內容補 `min-w-0`，手機 padding 降為 16px；導覽、按鈕、一般輸入框與 select 的觸控高度至少 44px。
- 薪資月份與待補歸屬在手機使用卡片版面；薪資詳情的項目明細、薪資設定、獎金級距、發薪區及加扣項 modal 改為可堆疊版面。
- 表單控制項統一使用既有 `surface`、`foreground`、`border` token，並以 `color-scheme` 支援原生 select／option 的 light／dark 呈現與 disabled 狀態。
- `eligible_sales_count` 改標示為「獎金級距計入台數」；未啟用獎金時不再顯示升級提示；`money_entry_source_review` 改顯示系統管理處理說明，不再連到無法修正來源的車輛頁。
- 待補歸屬頁新增獨立 loading state，API 尚未完成時不顯示 empty state。
