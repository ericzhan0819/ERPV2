- Vehicle／MoneyEntry Filter 由 URL 保存，支援 reload、上一頁／下一頁、分頁與 Mobile Drawer；`sold_month` 使用台北月份半開區間。
- App Shell、Mobile Sidebar、light／dark mode 與 iPhone Safari Safe Area 完成收尾，沒有新增 Header 快捷功能、通知或 Footer。
- `UI.md`、`backend/API.md`、`README.md`、`CLAUDE.md`、`docs/current-state.md`、Smoke Report 與 PLAN 已同步。
