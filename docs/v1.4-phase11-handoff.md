Mobile Sidebar 與 overlay 改為 App Shell 內的 viewport-height absolute layer，避開 fixed overlay 對 Safari 上下瀏覽器區域造成的 tint 殘留。Sidebar 開啟時由 `ThemeProvider` 將 `html`、`body`、`#root` 與 `theme-color` 同步為主題的 50% 黑色合成色，關閉後恢復正式 light／dark 底色；首次載入、主題切換、BFCache、頁面重新顯示與旋轉仍共用同一同步流程。
